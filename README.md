# 🛠️NINETO6 사이드프로젝트 만들기

## 시작하기 전에...

React StompJs 를 이용해 실시간 채팅 서비스를 구현 해보려 함

> 현재 `Front-End` 와 `Back-End` 는 다른 환경에서 개발하고 있음

## 요구사항

이를 위해 StompJs 의 함수 사용법을 알아야 함
또한 해당 destiny(목적지) 에 해당하는 url 이나 ip 번호를 지정해주어야 함
